# Installing

[PHP](https://php.net) 5.3+ and [Composer](https://getcomposer.org) are required.

In your composer.json, simply add `"league/factory-muffin": "~2.0"` to your `"require-dev"` section:
```json
{
    "require-dev": {
        "league/factory-muffin": "~2.0"
    }
}
```
